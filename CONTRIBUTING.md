# Contributing

OpenLLM Audit is in early development. Small, focused contributions are welcome.

Useful contributions include:

- new local runtime adapters
- new test profiles
- documentation improvements
- example local setups
- bug reports from real local testing
- hardening suggestions that are clear and practical

## Development Notes

Please keep changes small and easy to review.

Before opening a pull request, include:

- what changed
- why it helps
- how it was tested

Avoid adding cloud dependencies to the audit workflow. The main goal is to keep audit runs local.
