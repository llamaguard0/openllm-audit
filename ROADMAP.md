# Roadmap

This roadmap describes the first planned release of OpenLLM Audit.

## Milestone 1: Threat Model and Test Design

- define local LLM threat model
- define test profile format
- draft initial prompt and input tests
- document safe-use rules

## Milestone 2: CLI and Local Adapters

- build the first CLI runner
- add Ollama adapter
- add llama.cpp adapter
- add local HTTP API adapter
- produce JSON output

## Milestone 3: RAG and Runtime Checks

- add basic RAG abuse tests
- track latency, memory use, crashes, and timeouts
- improve result repeatability
- add first risk labels

## Milestone 4: Reports and Hardening Guidance

- add HTML report output
- add practical hardening suggestions
- add example local setups
- improve documentation

## Milestone 5: First Public Release

- clean install path
- add usage examples
- add contributor guide
- publish release notes
